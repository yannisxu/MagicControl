# Magic Control 需求文档

## 项目概述

**Magic Control** 是一款基于手势识别的 macOS 桌面应用，通过摄像头捕捉用户手势，实现无触控的翻页和激光笔控制功能。适用于演示、教学等场景。

---

## 技术栈

- **前端**: React + TypeScript + Vite
- **后端**: Tauri 2.0 (Rust)
- **手势识别**: MediaPipe Hands
- **平台**: macOS

---

## 核心功能

### 1. 手势识别

| 手势 | 动作 |
|------|------|
| 捏合拖拽（Pinch & Drag） | 左/上滑上一页，右/下滑下一页（松手触发） |
| 握拳长按 1.5s | 切换模式（激光笔 → 放大镜 → 关闭） |

### 2. 显示模式

- **激光笔模式** - 显示红色圆点，带尾迹效果，跟随食指移动
- **放大镜模式** - 显示蓝色圆圈区域
- **关闭模式** - 隐藏指示器

### 3. 系统特性

- 系统托盘运行，无 Dock 图标
- 全屏透明覆盖，不阻挡鼠标点击
- 实时光标平滑跟踪

---

## 权限要求

- 摄像头权限
- 辅助功能权限（用于模拟按键）
